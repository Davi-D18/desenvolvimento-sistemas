# README - Curso: Técnico em Desenvolvimento de Sistemas (SENAI)

Descrição
--------
Este repositório armazena as atividades e materiais do curso Técnico em Desenvolvimento de Sistemas (SENAI). Cada branch corresponde a um módulo específico do curso, permitindo organizar os trabalhos, exercícios e entregas de forma independente.

Estrutura
------------------
- /Materia-A/
  - /atv/          ← atividades (exercícios, trabalhos)
- /Materia-B/
  - /atv/
  - ...
- README.md

Organização das Branches
-----------------------
- main            ← branch principal com informações gerais
- Introdutorio    ← Atividades relacionados ao modulo introdutório
